# Architecture reviews

Point-in-time architecture reviews of tagged releases, kept as engineering
history alongside `DESIGN.md` and `taskforge/CONTRACTS.md`. Each review records
not only what to change but what to keep deliberately minimal — the
**Invariants to protect** and **Rejected** sections are as load-bearing as the
findings.
